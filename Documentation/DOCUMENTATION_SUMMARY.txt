## 🎯 Challenge Details (From Analysis)

### Task
**ImagingIQ CXR Triage Classification**
- Classify chest X-rays into 4 severity levels (p0, p1, p2, p3)
- Urgency-based triage system for patient assessment
- ~8000 test images from 100+ patients

### Model
- **Architecture**: ResNet-18 (Deep CNN)
- **Input**: 224×224 RGB images
- **Output**: 4-class probability distribution
- **Pre-trained**: Checkpoint provided (models/best.pt)

### Dataset
- **Total Images**: 8000+ CXR radiographs
- **Resolution**: 2500×2048 pixels (original)
- **Format**: PNG (grayscale medical images)
- **Metadata**: Age, gender, view position, pixel spacing, follow-up index
- **Structure**: Longitudinal studies (0-62 follow-ups per patient)

## 🚀 Quick Start Guide

### 1. Setup (One Time)
```bash
cd yolov11
python -m venv .venv
.venv\Scripts\activate  # Windows
source .venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
```

### 2. Run Inference
```bash
python predict.py \
  --images_dir test_public \
  --metadata_csv metadata.csv \
  --checkpoint models/best.pt \
  --out_csv outputs/test_public_predictions.csv
```

### 3. Check Results
```bash
head outputs/test_public_predictions.csv
# image_id,p0,p1,p2,p3
# 00000177_000.png,0.92,0.05,0.02,0.01
```
